Disk2: 샘플 코드 CD-ROM
--------------------------------------------------------------------------------

이 텍스트 파일은 CD-ROM 「Disk2」에 대해여 소개하고 있습니다.

■수록 내용에 대해
본 CD-ROM은 Windows용입니다.

본서에서 해설하고 있는 Visual Studio.NET (2002) 용 프로젝트 파일과 소스코드, 및 실행형식 파일(. exe)을 수록하고 있습니다.

각 폴더명의 숫자는 본서의 Chapter번호와 같습니다.
main.exe는 각 절에서 해설하고 있는 응용 프로그램입니다.

샘플 코드를 컴파일 시키려면 Visual C++.NET, Visual Studio.NET 등의 개발 환경이 필요합니다.
CD-ROM에는 이러한 개발툴이 수록되어 있지 않습니다.

※수록되어 있는 프로젝트용 파일을 「Visual Studio.NET 2003」으로 읽어들이면 오래된 형식의 파일이라는 경고 대화상자가 나오는 일이 있습니다. 
이런경우에는 대화상자에서 하라는 대로 현재 버젼으로 변환한뒤 이용하십시오.

개발 시에는 CD-ROM의 내용을 각 프로젝트 폴더마다, 하드 디스크에 복사 해서 이용해 주십시오. 
Windows XP이외의 OS에서는 복사 한 파일에 읽기전용 속성이 붙어있으므로, 이것들을 해제하고 나서 이용해 주십시오.

main.exe를 다른 시스템에서 실행하려면 동일한 폴더에 수록되어 있는 텍스쳐파일(. bmp)이나 X파일(. x) 등 관련파일도 동시에 필요합니다.
